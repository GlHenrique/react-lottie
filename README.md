# package build scripts

Build the app
`yarn dev`

Deploy to GH Pages 
`yarn deploy-example`

Publish to NPM
`npm publish --access=public`

See exported `Lottie`, `ReactLottieConfig`, `ReactLottieOwnProps`, `ReactLottiePlayingState` for info.
